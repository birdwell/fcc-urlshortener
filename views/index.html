FCC: URL Shortener Microservice

  • If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When I visit that shortened URL, it will redirect me to my original link.

Example Creation Usage

https://puzzled-meadow.glitch.me/new/http://www.nba.com https://puzzled-meadow.glitch.me/new/https://www.google.com

Example Response:

{"original_url":"http://www.nba.com","short_url":"https://puzzled-meadow.glitch.me/ynm3itc","_id":"5a179485334d756bf0267161"}

Usage

https://puzzled-meadow.glitch.me/ynm3itc
